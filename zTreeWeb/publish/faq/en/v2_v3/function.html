zTree v2.6 Method VS v3.0 Method

  • 1. Overview

    • zTree v3.0 - zTree v2.6: Initialization method has been slightly modified, in addition there are several method increases the trigger callback functions, easy to reduce redundant code.

  • 2. Detailed

    • It is arranged order by the zTree v3.0 structures. Because of limited space and time, please view the API documentation for specific instructions.
    • Method about $.fn.zTree
      ┌ $.fn.zTree.init(jQueryObj, setting, treeNodes) (v3.0 new)
      └ jQueryObj.zTree(setting, zTreeNodes) (v2.6)

      $.fn.zTree.getZTreeObj(treeId) (v3.0 new) --- For get zTree objects, easy to use.
      $.fn.zTree._z (v3.0 new) --- This object expose zTree’s private methods, and easy for advanced users. Mainly for the production of zTree auxiliary plug-in
